About Quick Message Composer

		
No version number

		
This is a demo application for the NJTools.JS library (and technically also Structure.CSS framework) written by Nathan Jackson.

		
If you find yourself using this seriously it has the following keyboard shortcuts when open:

		

		

			Key combination
			Description
		

		

			Ctrl + Shift + <
			Change page
		

		

			Ctrl + Shift + >
			Change page
		

		

			Ctrl + Shift + ?
			Snap to page first input
		

		

			Ctrl + Shift + C
			Copy current page preview
		

		

			Ctrl + Shift + Z
			Clear page and snap to first input
		

		

			Ctrl + Shift + R
			Reset app, return to first page and snap to first input
		

		

			Ctrl + Shift + A
			Bring up this About dialog
		

		

			Ctrl + Shift + S
			Bring up the Settings dialog
